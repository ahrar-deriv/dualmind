# Chat App

A Flutter-based chat application that provides real-time messaging capabilities.

## Features

- Real-time messaging
- User authentication
- Message history
- Clean and intuitive UI
- Cross-platform support (iOS, Android, Web)

## Getting Started

### Prerequisites

- Flutter SDK (latest version recommended)
- Dart SDK
- Android Studio / Xcode (for mobile deployment)
- Firebase account (for backend services)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/chat_app.git
   ```

2. Navigate to the project directory:
   ```
   cd chat_app
   ```

3. Install dependencies:
   ```
   flutter pub get
   ```

4. Copy the environment file and configure it:
   ```
   cp .env.example .env
   ```
   
5. Update the `.env` file with your configuration values

6. Run the app:
   ```
   flutter run
   ```

## Environment Configuration

The app uses environment variables for configuration. See `.env.example` for required variables.
